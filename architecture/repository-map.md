# Repository Map

## Repos de runtime

- `api-gateway`: entrada publica y politicas cross-cutting.
- `bff-mobile`: composicion para app mobile.
- `bff-backoffice`: composicion para panel administrativo.
- `microservice-users`: identidad, perfil y contexto de usuario.
- `microservice-quizz`: generacion y gestion de quizzes.
- `microservice-wordpass`: funcionalidad wordpass.
- `ai-engine`: capacidades AI y servicios de inferencia.

## Repos de plataforma

- `contracts-and-schemas`: OpenAPI, JSON Schema y eventos versionados.
- `shared-sdk-client`: SDKs multi-lenguaje derivados de contratos.
- `platform-infra`: IaC y despliegues por entorno.
- `observability-platform`: stack de metricas, logs, traces y alertado.
- `docs`: arquitectura, guias operativas y roadmap.
- `secrets`: catalogo de variables sensibles y convenciones de gestion de secretos.
